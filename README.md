# 📧 Spam Email Prediction using K-Nearest Neighbors (KNN)

## 📌 Project Overview

This project is a Machine Learning web application that predicts whether an email is **Spam** or **Not Spam** using the **K-Nearest Neighbors (KNN)** classification algorithm.

The application is built using **Python**, **Scikit-learn**, and **Streamlit**.

---

## 🚀 Features

- Predicts whether an email is Spam or Not Spam.
- Interactive Streamlit web application.
- Machine Learning model trained using KNN.
- Data preprocessing and feature scaling using StandardScaler.
- Fast and simple user interface.

---
